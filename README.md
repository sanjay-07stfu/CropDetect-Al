# CropDetect-Al
🌿 An intelligent web-based system for detecting crop leaf diseases using Convolutional Neural Networks (CNN) and image processing. Users can upload leaf images, and the model predicts disease type with high accuracy. Built using Python, Deep Learning, and Flask for real-time agricultural assistance.

## 📌 Project Overview
CropDetect-Al is a deep learning-based web application that detects plant leaf diseases using Convolutional Neural Networks (CNN) and image processing techniques.  

The system allows users to upload a leaf image and receive real-time disease prediction with confidence score.

This project aims to assist farmers and agriculture students by providing quick and accurate plant disease detection.

## 🚀 Features
- 🌿 Upload plant leaf images
- 🧠 CNN-based disease classification
- 📊 Prediction with confidence score
- 🌐 Web-based interface (Flask)
- ⚡ Fast and user-friendly design

## 🧠 How It Works
1. User uploads a plant leaf image.
2. Image is preprocessed (resizing, normalization).
3. Trained CNN model predicts disease class.
4. Result is displayed with confidence percentage.
