## ❓ FREQUENTLY ASKED QUESTIONS

**Q: Do I need a trained model to start?**
A: No! Demo mode is active. The system randomly selects diseases until you provide a trained model.

**Q: How do I add my trained model?**
A: Place it at `model/crop_model.h5` and restart the app. System auto-detects it.

**Q: Works only locally?**
A: Currently yes. Docker config and deployment guides included for going live.

**Q: Can I add more diseases?**
A: Yes! Edit `DISEASE_INFO` dictionary in app.py and add new diseases.

**Q: Mobile compatible?**
A: Fully responsive - works great on phones and tablets.

**Q: Can I change colors/design?**
A: Easily! Edit `static/css/style.css` for styling customization.

**Q: How to export detection history?**

**Q: Database location?**
